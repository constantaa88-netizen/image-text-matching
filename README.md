## 📌 О проекте

Система решает задачу **кросс‑модального поиска**: по текстовому запросу (например, *«две собаки бегут по полю»*) найти наиболее релевантное изображение.

**Как работает:**  
Изображения и тексты кодируются в общее векторное пространство с помощью предобученных нейросетей (ResNet‑18 и DistilBERT). Затем небольшая нейросеть оценивает степень соответствия пары (вероятность от 0 до 1). После обучения для любого запроса вычисляются оценки для всех изображений галереи и возвращается лучший кандидат.

---

## 🧠 Архитектура модели

| Компонент | Детали |
|-----------|--------|
| **Image encoder** | ResNet‑18, предобучен на ImageNet, извлекаются признаки с выхода `flatten` (512‑d) |
| **Text encoder** | DistilBERT‑uncased + mean pooling (768‑d) |
| **Fusion MLP** | 3 скрытых слоя (1024 → 512 → 256), ReLU, Dropout(0.3), выход 1 нейрон |
| **Loss** | `BCEWithLogitsLoss` (бинарная кросс‑энтропия) |

---
## 📊 **Данные**

Объём: ~5 800 размеченных пар «изображение–текст».

Источники разметки:

Краудсорсинг – доля людей, подтвердивших соответствие.

Эксперты – средняя оценка трёх специалистов (шкала 1–4).

Целевая переменная: взвешенная сумма 0.7 * экспертная_вероятность + 0.3 * крауд‑доля, приведённая к интервалу [0, 1].

Конфиденциальность: автоматическое удаление изображений с детьми (по ключевым словам в запросе).

## 📈 **Результаты**

| Модель | Validation RMSE |
|-----------|--------|
|**Линейная регрессия (baseline)**|	0.486
|**Обучаемая MLP**|	0.708

✅ MLP снижает ошибку на ~37% по сравнению с линейной регрессией, что подтверждает эффективность нелинейного объединения мультимодальных признаков.

## 🔧 Возможные улучшения

**Текстовый энкодер** – заменить на `sentence-transformers/all-MiniLM-L6-v2` (компактнее, быстрее, качественнее).

**Кросс‑внимание** – вместо конкатенации использовать взаимодействие через cross‑attention (как в CLIP).

**Hard negative mining** – для лучшего разделения похожих, но нерелевантных пар.

**Аугментация изображений** – повышение устойчивости модели.

**Дистилляция** – для ускорения инференса в production.
